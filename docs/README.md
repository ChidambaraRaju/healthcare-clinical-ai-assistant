## Overview

The Healthcare Clinical AI Assistant is an AI-powered platform designed to help healthcare professionals with clinical documentation, medical code suggestions, and document summarization. It is built with HIPAA compliance in mind and includes enterprise-grade security features.

## Architecture

The application consists of three main services:

- **Backend API**: FastAPI-based REST API for business logic and data management
- **Frontend**: Next.js web application for user interface
- **ML Service**: Python-based AI inference service for clinical NLP tasks

## Features

- AI-powered clinical note generation
- ICD-10 and CPT code suggestions
- Document summarization
- HIPAA-compliant data handling
- Role-based access control
- Comprehensive audit logging
